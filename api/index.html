MySQL 8.0的新增功能探索

自从2005年Oracle收购InnoDB存储引擎开发商Innobase伊始，MySQL的命运已然注定。

随着MySQL 5.6开始的源码重构，MySQL已经驶上了快车道，MGR给了PXC致命一击，而Clone Plugin的推出宣判了MariaDB的死刑。MariaDB，或许只能拿来缅怀曾经的青春吧。
